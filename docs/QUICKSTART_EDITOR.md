# AI Editor 快速开始指南

## 🚀 5 分钟快速开始

### 1. 确保依赖已安装

```bash
# 如果是首次使用，需要安装依赖
npm install --legacy-peer-deps
```

### 2. 配置环境变量

在项目根目录创建或编辑 `.env.local` 文件：

```bash
# 添加你的 OpenAI API Key
OPENAI_API_KEY=sk-your-api-key-here
```

### 3. 启动开发服务器

```bash
npm run dev
```

或使用 bun：

```bash
bun dev
```

### 4. 访问编辑器

1. 打开浏览器访问 `http://localhost:3000`
2. 点击左侧边栏的 **"AI Editor"** 菜单项
3. 开始使用！

## 📝 基本使用

### 编写文本

直接在编辑器中输入文本，就像使用 Word 或 Google Docs 一样。

### 格式化文本

使用顶部工具栏的按钮：

- **B** - 粗体
- **I** - 斜体
- **U** - 下划线
- **S** - 删除线
- **H1, H2, H3** - 标题
- 列表图标 - 创建列表
- 引号图标 - 引用块

### 使用 AI 生成内容

1. **方法 1**: 选中一段文本
   - 选择要扩展的文本
   - 点击工具栏的 **"AI 生成"** 按钮
   - AI 会根据选中的内容继续生成

2. **方法 2**: 从提示词开始
   - 输入一个简短的提示，比如 "写一篇关于人工智能的文章"
   - 选中这段提示
   - 点击 **"AI 生成"**
   - AI 会基于提示生成完整内容

### 保存和导出

- **保存**: 点击 **"保存"** 按钮（当前会在控制台输出内容）
- **导出**: 点击 **"导出"** 按钮下载 HTML 文件

## ⌨️ 键盘快捷键

- `Cmd/Ctrl + B` - 粗体
- `Cmd/Ctrl + I` - 斜体
- `Cmd/Ctrl + U` - 下划线
- `Cmd/Ctrl + Z` - 撤销
- `Cmd/Ctrl + Shift + Z` - 重做

## ❓ 常见问题

### Q: AI 生成按钮点击后没反应？

**A**: 检查以下几点：

1. 确保已设置 `OPENAI_API_KEY` 环境变量
2. 确保已选中文本
3. 检查浏览器控制台是否有错误
4. 确认 OpenAI API 配额充足

### Q: 如何更改 AI 模型？

**A**: 编辑 `/app/api/editor/generate/route.ts`:

```typescript
model: openai('gpt-4') // 改为 gpt-4 或其他模型
```

### Q: 生成的内容太短/太长？

**A**: 暂时无法在 UI 中调整，但可以修改 API 代码中的 temperature 参数：

```typescript
temperature: 0.7 // 0.0-1.0, 值越高越有创造性
```

### Q: 如何保存到数据库？

**A**: 当前保存功能仅输出到控制台。要实现数据库保存，需要：

1. 创建数据库表
2. 实现保存 API
3. 在 `app/editor/page.tsx` 中实现 `handleSave` 函数

参考 [NOVEL_INTEGRATION.md](./NOVEL_INTEGRATION.md#添加数据库持久化) 了解详细步骤。
